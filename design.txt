 2️⃣ 知识图谱模块（核心！！！）

👉 用 **Neo4j**

 1. 节点设计（Node）

* Knowledge（知识点）
* Course（课程）
* Chapter（章节）

 2. 关系设计（Relation）

* `PREREQUISITE`（前置知识）
* `BELONG_TO`（属于课程）
* `SIMILAR_TO`（相似知识）
* `HAS_CHAPTER`

---

 3. 示例（Cypher）

```cypher
CREATE (a:Knowledge {name: "线性回归", difficulty: 3})
CREATE (b:Knowledge {name: "概率论", difficulty: 2})
CREATE (b)-[:PREREQUISITE]->(a)
```

---

 4. Spring Boot接入

用：

* Spring Data Neo4j

```java
@Node("Knowledge")
public class Knowledge {
    @Id @GeneratedValue
    private Long id;
    private String name;
    private Integer difficulty;
}
```

 3️⃣ 个性化学习路径推荐模块（重点难点🔥）

这个模块是你论文“含金量最高”的地方。

---

 思路：三层融合推荐

---

 ✅ 第一层：图搜索（保证知识逻辑）

用：

* BFS / DFS
* 或最短路径算法

👉 Neo4j直接支持：

```cypher
MATCH path = shortestPath(
 (start:Knowledge {name:"基础"})
 -[:PREREQUISITE*]->
 (end:Knowledge {name:"机器学习"})
)
RETURN path
```

---

 ✅ 第二层：难度匹配

规则：

* 当前用户level = 2
* 推荐 difficulty = 2~3

```text
score = |knowledge.difficulty - user.level|
```

---

 ✅ 第三层：兴趣推荐（简单版）

两种方案（任选一个就够毕业）：

 ✔ 内容相似度（推荐简单）

* 标签匹配
* TF-IDF

 ✔ 协同过滤（加分项）

* 用户A学了什么
* 相似用户推荐

---

 ✅ 最终融合算法（论文亮点✨）

```text
FinalScore = 
  α * 路径合理性 +
  β * 难度匹配 +
  γ * 兴趣相似度
```
